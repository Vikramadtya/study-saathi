Problem Statement:
Given an array arr of integers. A peak element is defined as an element greater than both of its neighbors.
Formally, if arr[i] is the peak element, arr[i - 1] < arr[i] and arr[i + 1] < arr[i].

Find the index(0-based) of a peak element in the array. If there are multiple peak numbers, return the index of any peak number.


Candidate Solution:

```java
class Solution {
  public int findPeakElement(int[] arr) {
    int left = 0, right = arr.length - 1;

    while (left <= right) {

      int mid = (right - left) / 2 + left;

      int lval = mid - 1 >= 0 ? arr[mid - 1] : Integer.MIN_VALUE;
      int rval = mid + 1 < arr.length ? arr[mid + 1] : Integer.MIN_VALUE;

      if (lval == rval) {
        int c = 2;
        while (mid - c >= left && mid + c <= right && lval == rval) {
          lval = mid - 1 >= 0 ? arr[mid - c] : Integer.MIN_VALUE;
          rval = mid + 1 < arr.length ? arr[mid + c] : Integer.MIN_VALUE;
          c++;
        }
      }

      if (lval < arr[mid] && rval < arr[mid]) return mid;

      if (lval > rval) {
        right = mid - 1;
      } else {
        left = mid + 1;
      }
    }

    return -1;
  }
}
```
