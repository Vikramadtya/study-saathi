### INPUT

Problem Statement:

Given an array of integers nums of unique elements. Return all possible subsets (power set) of the array.

Do not include the duplicates in the answer.

Candidate Solution:

```java
class Solution {
    public List<List<Integer>> powerSet(int[] nums) {

        List<List<Integer>> result = new ArrayList<>();
        boolean [] included = new boolean[nums.length];
        powerSetCreator(result,included,nums,0);
        return result;
    }

    public void powerSetCreator(List<List<Integer>> result, boolean[] included, int[] nums, int index) {

        if(index == nums.length) {
            List<Integer> current = new ArrayList<>();
            for(int i = 0; i < nums.length; ++i) {
                if(included[i]) current.add(nums[i]);
            }
            result.add(current);
            return;
        }

        // include
        included[index] = true;
        powerSetCreator(result,included,nums,index+1);

        included[index] = false;
        powerSetCreator(result,included,nums,index+1);
    }
}
```

Claimed Complexity:

Space Complexity : O(2^n)
Time Complexity : O(2^n \* n)
